EASY-BUS


Reto:

Crear un producto que mejore la experiencia de los usuarios en las empresas de buses

Solución:

Easy-Bus, es una aplicación permite comprar o reservar pasajes, a cualquier parte del Perú, a un precio normal cuando se acerca un feriado o alguna festividad, evitando pagar un precio extra por el pasaje.

Herramientas:

  • Figma
  • Marvel
  • Content-prototypy
  • User-person
  • Storyboard
  • Storytelling

LYFT


Reto:

Desarrollar una web-app que replique el sitio de Lyft y que cumpla con los pasos necesarios para que el usuario pueda registrarse.

Solución:

Se maqueto la aplicación con una paleta de colores con los colores caracteristicos de Lyft y se implemento la funcionalidad del proceso de registro de un nuevo usuario y la verificación de los datos que se ingresa.

Herramientas:

  • Html5
  • Css3
  • Jquery
  • Bootstrap
  • Responsive-web
